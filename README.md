## Customization

### Mengubah Warna
Edit variable CSS di `styles.css`:
```css
:root {
    --primary-cyan: #00d4ff;   /* Warna biru muda utama */
    --dark-black: #1a1a1a;     /* Warna hitam gelap */
    --light-gray: #f5f5f5;     /* Warna abu-abu terang */
}
```

### Menambah Gambar Portofolio
1. Simpan gambar ke folder root dengan nama `portfolio[nomor].jpg`
2. Update HTML untuk menambah portfolio item baru

### Mengubah Konten
Edit konten langsung di `index.html` sesuai kebutuhan.
